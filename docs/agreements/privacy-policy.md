# 最佳干饭人 · 隐私政策

**生效日期**：2026 年 7 月 1 日
**版本号**：v1.0
**最后更新**：2026 年 7 月 1 日

---

## 引言

「最佳干饭人」（以下简称"本小程序"）由个人开发者运营，**严格遵守**《中华人民共和国个人信息保护法》《网络安全法》《数据安全法》等法律法规。

本政策旨在**清晰透明**地说明：
- 我们收集哪些信息
- 为什么收集
- 如何使用和存储
- 您拥有哪些权利

---

## 一、我们收集的信息

### 1.1 您主动提供的信息

| 信息类型 | 收集时机 | 用途 |
|---|---|---|
| **微信 OpenID** | 首次登录时 | 唯一标识您的账号 |
| **微信昵称、头像** | 您主动授权时 | 在搭子页和打卡页展示 |
| **打卡图片** | 您拍照或选图时 | 记录您的饮食内容 |
| **餐次 / 类别 / 备注** | 您提交打卡时 | 完整记录一次饮食 |
| **配对码** | 您发起或加入配对时 | 与搭子建立关联 |

### 1.2 系统自动收集的信息

| 信息类型 | 用途 |
|---|---|
| **设备型号、操作系统版本** | 兼容性适配 |
| **微信版本号** | 功能兼容性判断 |
| **网络状态** | 错误诊断（不保存原始报文） |

### 1.3 我们**不**收集的信息

- ❌ 您的真实姓名、身份证号、手机号
- ❌ 您的地理位置（不申请此权限）
- ❌ 您的通讯录
- ❌ 您的麦克风录音
- ❌ 您的微信好友列表
- ❌ 任何金融、支付信息

---

## 二、信息的使用

我们仅在**以下必要场景**使用您的信息：

1. **提供核心功能**：登录、打卡、配对、查看搭子数据
2. **数据统计**：每周工作日饮食统计、5 大食物类别分布
3. **内容安全**：调用微信官方接口检测违规图片和文字
4. **故障诊断**：崩溃日志分析（不包含您的打卡内容）

我们**承诺**：
- ✅ 不用于商业广告
- ✅ 不进行用户画像、个性化推荐
- ✅ 不向任何第三方出售或提供
- ✅ 不用于自动化决策

---

## 三、信息的存储

### 3.1 存储位置

- 您的打卡数据存储在**微信云开发**（腾讯云）数据库
- 您的图片存储在**微信云开发**云存储
- 存储区域：**中国境内**（符合《数据安全法》要求）

### 3.2 存储期限

| 数据类型 | 存储期限 |
|---|---|
| 打卡记录 | 直至您删除账号 |
| 图片 | 直至您删除对应打卡或账号 |
| 配对关系 | 直至您主动解除 |
| 内容安全审核日志 | **不少于 60 天**（法律法规要求） |
| 崩溃日志 | **不超过 30 天** |

### 3.3 安全措施

- 全程 HTTPS 加密传输
- 云开发数据库访问需鉴权
- 您的 OpenID 不会暴露给其他用户
- 搭子仅能查看**您授权范围内的**打卡数据

---

## 四、信息的共享

### 4.1 唯一共享场景：搭子配对

当您**主动**与某人配对成功后：
- 您的**昵称、头像**对搭子可见
- 您当天的**打卡状态**（已打卡 / 未打卡）对搭子可见
- 您的**打卡图片、餐次、类别**仅在您**主动展开详情**时对搭子可见

### 4.2 法律强制共享

以下情况我们**可能**依法共享：
- 配合司法机关调查
- 配合网信、公安等部门监管
- 应对紧急安全事件

## 五、您的权利

您对自己的信息**拥有充分权利**：

| 您的权利 | 如何行使 |
|---|---|
| **查看** | 进入小程序任意页面查看 |
| **修改** | 在"我的"页修改昵称、头像 |
| **删除单条打卡** | 长按打卡记录 → 删除 |
| **解除配对** | 在"搭子"页 → 解除配对 |
| **清空全部数据** | 在"我的"页 → 设置 → 清空所有数据 |
| **注销账号** | 在"我的"页 → 设置 → 注销账号（**不可恢复**） |
| **导出数据** | 在"我的"页 → 设置 → 导出我的数据 |

**响应时效**：收到您的请求后，**15 个工作日内**处理。

## 六、未成年人保护

1. 本小程序面向**14 周岁以上**用户
2. 我们**不会主动收集**14 周岁以下未成年人的个人信息
3. 如您是未成年人，请在**监护人指导下**阅读本协议并使用本小程序
4. 如您是监护人，发现未成年人在未经同意下使用了本小程序，请通过"十、联系方式"告知我们，我们会**立即删除**相关数据

## 七、Cookie 与类似技术

本小程序为**微信内嵌运行**，**不使用**传统浏览器的 Cookie、localStorage 等追踪技术。
我们仅使用微信小程序原生的**本地缓存**（`wx.setStorageSync`），用于：
- 保存登录态
- 缓存本周统计数据（提升加载速度）

您可在微信"我 → 设置 → 通用 → 存储空间"中**清除小程序缓存**。
