## 三、小程序简介（详细版，用于商店详情页）

```
最佳干饭人是一款朋友间 1:1 配对的趣味饮食记录小程序。

【核心功能】
• 拍照打卡：早/午/晚/加餐，4 个时间点不漏一顿
• 5 大食物类别：主食、蔬菜水果、肉蛋水产、汤粥饮品、甜品零食，让饮食结构一目了然
• 1:1 配对：与朋友私密配对，互相查看打卡，互相陪伴
• 每周总结：工作日 5 天统计，圆环+日历直观展示你的坚持
• 梦幻治愈：马卡龙色系，圆角胶囊，让每次打卡都是享受

【设计理念】
我们相信，习惯的养成离不开陪伴。最佳干饭人不是冰冷的打卡工具，
而是与朋友之间的温暖约定——按时吃饭、好好吃饭、一起变好。

【隐私承诺】
• 不收集位置、通讯录等敏感权限
• 数据仅在配对的搭子间可见
• 可随时解除配对、删除数据

【特别说明】
本小程序为个人开发者非盈利项目，仅供朋友间使用。
```

## 五、类目选择

**推荐**：工具 → 健康管理

| 备选类目 | 风险 |
|---|---|
| ✅ 工具 → 健康管理 | **推荐**，最合规 |
| ⚠️ 工具 → 效率 | 也可，但审核员可能要求证明 |
| ❌ 社交 | 100% 被拒 |
| ❌ 社区 | 100% 被拒 |
| ❌ 健康 → 医疗服务 | 需要资质，拒 |

**类目描述**（mp 后台填写）：

```
为用户提供个人饮食记录工具，支持 1:1 私密数据分享（非公开社区），
不提供医疗建议或诊断服务。
```

## 六、审核备注（给审核员看的）

```
本小程序为个人开发者非盈利项目，仅供朋友间使用。

【核心定位】
个人饮食记录工具，非社交平台。

【1:1 配对功能说明】
- 用户输入 6 位配对码与特定好友配对（需对方同意）
- 配对是 1:1 私密关系，非群组、非社区
- 用户可随时解除配对
- 仅展示对方主动提交的打卡内容（图片+文字+类别）

【为何选择"工具-健康管理"】
- 核心功能是"个人记录"，符合工具属性
- 健康管理类目下允许轻量级健康数据记录
- 我们不提供医疗建议，不涉及诊断

【内容安全保障】
- 所有图片经 imgSecCheck 检测
- 所有文字经 textSecCheck 检测
- 违规内容直接拒绝上传

【测试账号】
- 测试 openid: 已在云开发数据库预置
- 测试流程：登录 → 拍照打卡 → 配对 → 查看搭子
```
