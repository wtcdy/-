## 一、服务说明

「最佳干饭人」（以下简称"本小程序"）是一款由个人开发者运营的**非盈利**微信小程序，旨在帮助用户：
- 记录个人每日饮食情况（早 / 午 / 晚 / 加餐）
- 标记每次饮食所属的食物类别（主食 / 蔬菜水果 / 肉蛋水产 / 汤粥饮品 / 甜品零食）
- 通过 1:1 私密配对，与朋友互相查看对方打卡记录
- 查看个人与搭子的每周工作日饮食统计

本小程序**不提供**以下服务：
- 医疗建议、营养诊断、健康治疗
- 社交动态、群组聊天、内容发布
- 商业推广、广告投放、虚拟商品销售

---

## 二、用户行为规范

您在使用本小程序时，**同意并承诺**：

1. **真实记录**：所上传的打卡内容（图片、文字）均为本人真实饮食情况，不得使用他人图片或编造记录
2. **文明用语**：备注、昵称等用户生成内容（UGC）不得包含：
   - 法律法规禁止的内容
   - 暴力、色情、赌博、毒品、邪教等不良信息
   - 侮辱、诽谤、骚扰他人的内容
   - 任何形式的商业广告或推广信息
3. **保护隐私**：与搭子配对时，应取得对方明确同意；不得擅自公开他人打卡记录
4. **合法使用**：不得利用本小程序从事任何违法违规活动

## 三、内容审核

为符合《网络安全法》《个人信息保护法》等法律法规要求：

- 您上传的**图片**将经过微信官方 `imgSecCheck` 接口进行安全检测
- 您填写的**文字**将经过微信官方 `textSecCheck` 接口进行安全检测
- 检测不通过的内容将被拒绝上传，违规严重者将被封禁账号
- 审核记录将依法留存，**不少于 60 天**

---

## 四、隐私保护

我们**高度重视**您的隐私保护，详见《隐私政策》。核心承诺：

1. **不收集敏感信息**：不获取您的位置、通讯录、摄像头（仅拍照时）、麦克风等敏感权限
2. **最小化存储**：仅保存您主动提交的打卡数据
3. **不向第三方分享**：您的数据仅在小程序内展示给已配对的搭子
4. **您可随时删除**：解除配对、删除打卡、清空账号数据均可在小程序内操作
